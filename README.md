# SAGIP PILIPINAS 2028 🇵🇭

Free, satirical voter-education games for the Philippines.

**No real names appear in these games. Only the tactics are real** — vote
buying (budol), bloc voting, fake news, dynasties, ghost projects — all
documented, all receipted in-game.

## The games

| Game | What it is |
|---|---|
| [Boss Edition](boss/) | Face the ten bosses of budol — and the final boss, THE SURRENDER. Tagalog + English. |
| [Sink or Swim](sagip/) | One campaign season, 25 chapters, a rising flood. The country swims or sinks on your answers. |
| [Church, Inc.](churchinc/) | Satire: play the cult leader and learn the machine from inside. English. |

## The one rule

Same questions, every candidate, every election — regardless of party,
surname, or church. Verify in real life: COMELEC · PPCRV · NAMFREL ·
COA reports · Senate/House records.

The ballot is secret. No organization can know how you voted — that is
law, not a promise.
